# Vue installation

## install nvm
```
brew install nvm
```
## install node version 15
```
nvm install 15
```
## install vue cli
```
npm install -g @vue/cli
```
## create project
```
vue create hello
```
## move into the project
```
cd hello
```
## install packages
```
vue add electron-builder

vue add vuetify

vue add eslint
```

# hello

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
